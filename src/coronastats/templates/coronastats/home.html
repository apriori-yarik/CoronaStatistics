COVIDSTATISTICS

CovidStatistics is a web app which shows all the info about Covid-19 - new and total cases, deaths, number of recovered people, some statistics to make the situation more clear. You can compare two or more countries as well as see predictions about the new/total cases, deaths and so on for the next few days/weeks/months.
